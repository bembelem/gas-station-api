# Gas Station Management API

Backend API для управления системой автозаправочных станций.  
Проект разработан в рамках учебной работы и демонстрирует
классический подход к проектированию серверной части
с использованием реляционной базы данных.

## Используемые технологии

- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy (ORM)
- Pydantic

API покрывает управление топливом, НПЗ, АЗС, клиентами,
резервуарами, продажами и аналитическими отчётами.

---

## Установка и запуск проекта

### 1. Установка зависимостей

Рекомендуемый способ (через `uv`):

```bash
uv sync
```
Альтернативный способ (через pip):
```bash
pip install -r requirements.txt
```

### 2. Настройка базы данных PostgreSQL

Необходимо создать базу данных PostgreSQL и указать параметры
подключения:

- host

- port

- database

- user

- password

Параметры могут быть заданы через файл .env
или напрямую в конфигурации SQLAlchemy.

### 4. Запуск сервера

Запуск через Python:

```bash
python src/main.py
```

Или напрямую через Uvicorn:

```bash
uvicorn backend.main:app --reload
```

После запуска сервер будет доступен по адресу:

http://127.0.0.1:8000
